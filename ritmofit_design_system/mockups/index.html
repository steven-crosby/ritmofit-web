What these mockups encode

  • Tempo is a token. --rf-bpm drives --rf-beat; the playing track's art pulses on the actual beat (Live + selected row only, suppressed under reduced-motion).
  • The numbers carry the brand. Every BPM and timecode is set in Martian Mono.
  • The class has a shape. The energy-arc ribbon renders intensity over time; height encodes the zone so it reads in grayscale.
  • Color never carries meaning alone. Intensity is always color + bar-count + label; cyan is the only "you can act on this" color; plasma magenta appears only at All-Out.
