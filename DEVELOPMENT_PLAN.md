## One-line project summary

RitmoFit is a Cloudflare-native full-stack TypeScript app for rhythm spin cycle instructors to plan classes, organize music metadata, choreograph tracks, add timestamped cues, and prepare future iOS live-mode payloads.

## Current priority

Build the foundation and Class Builder Timeline before implementing full music-provider integrations.

SoundCloud is the first real provider because it is RitmoFit’s key differentiator.
